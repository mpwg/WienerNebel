# AGENTS.md

## Arbeitsregeln

- Verwende deutsche Umlaute.
- Erstelle für jeden abgeschlossenen Task immer einen PR.
- Merge abgeschlossene Tasks immer per Squash Merge, sofern der User nichts anderes vorgibt.
- Ein Task gilt erst als vollständig abgeschlossen, wenn der PR erstellt und per Squash Merge abgeschlossen wurde oder der User diese Regel ausdrücklich aussetzt.
